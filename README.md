# 🧠 Notion AI Vocabulary Bot（中文说明）

---

## 项目简介

这是一个基于 **Notion + Google Gemini + GitHub Actions** 的自动化英语词汇学习系统。

项目目标是：

- 减少手动整理与复习的负担
- 通过 **主动回忆（Active Recall）** 提升记忆效果
- 使用自动化流程代替自律

---

## ✨ Quiz 系统说明

本项目包含一个自动生成选择题（MCQ）的 Quiz 系统，用于词汇复习。

### Quiz 的特点

- 每个单词 **只生成一次题目**
- 已生成的题目不会被自动覆盖
- 支持基于艾宾浩斯遗忘曲线的多次复习设计
- 题目由 Google Gemini 自动生成

---

## 📝 题目生成规则

当满足以下条件时，系统会自动为单词生成 Quiz：

- 单词存在于 Notion 数据库中
- `Question` 字段为空（说明尚未生成题目）
- 单词达到可复习状态

一旦生成题目，系统不会再次修改该题目内容。

---

## 📊 Notion 字段说明

| 字段名     | 类型        | 说明             |
| ---------- | ----------- | ---------------- |
| Question   | 文本        | 自动生成的选择题 |
| Answer Key | 文本 / 选择 | 正确答案         |
| My Answer  | 选择        | 用户选择的答案   |
| Check      | 状态        | 是否答对         |
| Quiz Due   | 勾选框      | 是否需要复习     |
| Last Quiz  | 日期        | 上一次作答时间   |

---

## 🤖 自动化流程

Quiz 系统通过 **GitHub Actions** 自动运行：

1. 查询 Notion 中符合条件的单词
2. 调用 Gemini API 生成题目
3. 将题目与答案写回 Notion
4. 全程无需手动触发

所有密钥均通过 GitHub Secrets 管理。

---

## 🧩 设计理念

- **主动回忆优于被动记忆**
- **系统自动化优于强迫自律**
- **稳定题目优于频繁变化**

本项目是一个长期可维护的个人学习系统，而非一次性脚本。
